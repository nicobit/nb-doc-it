## Contribution Templates

When creating new issues, use the GitLab issue templates:

- **Bug report** — Use the `bug` template for defects.
- **Feature request** — Use the `feature-request` template for new capabilities.
- **Documentation update** — Use the `docs` template for portal changes.

    User stories, acceptance criteria, and sprint-level tasks live exclusively in GitLab. This portal holds only durable product context.
