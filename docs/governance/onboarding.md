# Onboarding Guide

Day 1 checklist for new team members joining the Smart City.

## Access Requests

- [ ] GitLab: Request access to the product group and relevant repositories.
- [ ] SharePoint: Request access to the product documentation site.
- [ ] Azure: Request access to non-production environments.
- [ ] Monitoring: Request access to dashboards (Datadog, Grafana, Application Insights).
- [ ] Communication: Join the team channels (Slack/Teams).

## Local Environment Setup

1. Clone the relevant repositories from the [Module Registry](../engineering/registry.md).
2. Follow the `README.md` in each repository for local setup instructions.
3. Verify you can run the application locally and execute the test suite.

- [ ] Complete all access requests.
