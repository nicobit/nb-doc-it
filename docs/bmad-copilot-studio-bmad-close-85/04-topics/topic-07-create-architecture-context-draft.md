# Topic 07 — Create Architecture Context Draft

## Trigger phrases

- Create architecture context
- Review architecture impact
- Extract technical constraints

## Goal

Create an architecture context draft for BMAD Architect review in VS Code.

## Steps

1. Extract known systems, integrations, data, security, NFRs, operational concerns.
2. Separate confirmed constraints from assumptions.
3. Do not invent repo-specific implementation design.
4. Identify questions for the BMAD Architect.

## Output

Use `06-output-templates/06-architecture-context-draft-template.md`.
