# Topic 02 — Create Project Brief

## Trigger phrases

- Create project brief
- Produce the brief
- Summarize the initiative for BMAD

## Goal

Create a BMAD-ready project brief draft.

## Steps

1. Summarize the initiative.
2. Document business goals, users, systems, scope, out-of-scope, constraints.
3. Add source evidence and confidence.
4. List risks, assumptions, open questions.

## Output

Use `06-output-templates/01-project-brief-template.md`.
