# Topic 06 — Create Story Candidates

## Trigger phrases

- Create story candidates
- Generate candidate stories
- Break epics into draft stories

## Goal

Create candidate stories for BMAD validation. Do not create implementation-ready stories.

## Steps

1. Use epic candidates and PRD draft.
2. Create story candidates with draft acceptance criteria.
3. Add business value, assumptions, open questions, dependencies, and architecture notes.
4. Mark each story as Candidate / Not implementation-ready.
5. Include Do Not Implement Until criteria.

## Output

Use `06-output-templates/05-story-candidates-template.md`.
