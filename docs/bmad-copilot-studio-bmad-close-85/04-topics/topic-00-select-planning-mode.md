# Topic 00 — Select Planning Mode

## Trigger phrases

- Select planning mode
- What mode should this initiative use?
- Classify this BRS
- Is this quick, standard, or enterprise?

## Goal

Classify the initiative as Quick, Standard, or Enterprise before generating the handoff package.

## Steps

1. Read the source material or user description.
2. Assess scope, risk, number of systems, regulatory/security impact, and stakeholder complexity.
3. Select one mode.
4. Explain why.
5. List outputs required for that mode.

```markdown
# Planning Mode Selection

## Selected Mode
Quick / Standard / Enterprise

## Evidence

## Additional Caution Areas
```
