# Topic 05 — Create Epic Candidates

## Trigger phrases

- Create epic candidates
- Generate epics
- Split into epics

## Goal

Create outcome-based epic candidates from the PRD draft and capability map.

## Steps

1. Group capabilities and requirements into delivery outcomes.
2. Avoid creating epics that are only UI pages or technical layers.
3. Link epics to source capabilities and business outcomes.
4. Add dependencies, risks, assumptions, and open questions.

## Output

Use `06-output-templates/04-epic-candidates-template.md`.
