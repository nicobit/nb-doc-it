# Topic 04 — Create PRD Draft

## Trigger phrases

- Create PRD draft
- Produce PRD
- Turn BRS into PRD draft

## Goal

Create a PRD draft for later BMAD validation.

## Steps

1. Use the project brief and capability map.
2. Define business problem, desired outcomes, personas, scope, requirements, NFRs.
3. Include risks, assumptions, and open questions.
4. Mark status as Draft / Requires BMAD validation.

## Output

Use `06-output-templates/03-prd-draft-template.md`.
