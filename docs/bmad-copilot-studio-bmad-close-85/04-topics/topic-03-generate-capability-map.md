# Topic 03 — Generate Capability Map

## Trigger phrases

- Generate capability map
- Extract capabilities
- Identify business capabilities

## Goal

Create a capability map that separates stable business capabilities from implementation features or UI pages.

## Steps

1. Extract business capabilities.
2. Group related capabilities.
3. Link each capability to source evidence.
4. Add confidence and open questions.
5. Identify likely candidate epics but do not finalize them.

## Output

Use `06-output-templates/02-capability-map-template.md`.
