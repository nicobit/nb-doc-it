# Runbook — First Pilot

1. Choose one anonymized BRS.
2. Place it in `03-input/brs/`.
3. Ask the agent to select planning mode.
4. Run full handoff package generation.
5. Save outputs under `05-output/bmad-handoff/<initiative-id>/`.
6. Copy outputs into a VS Code repo under `docs/bmad-handoff/`.
7. Paste `08-bmad-vscode-handoff-prompt.md` into BMAD in VS Code.
8. Check how much rework BMAD needs.
9. Tune templates and instructions based on gaps.
