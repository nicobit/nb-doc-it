# Evidence and Traceability Rules

Every major artefact should preserve a path back to source material.

## Evidence labels

Use:

- Source Document
- Section / Page / Heading if known
- Extracted Fact
- Interpretation
- Confidence

## Requirement classification

Classify requirements as:

- Confirmed: explicitly present in source material.
- Inferred: reasonably derived from source material.
- Assumption: needed to proceed but not confirmed.
- Open Question: must be answered before final delivery or implementation.

## Traceability expectations

For Enterprise mode, all major capabilities, epics, and story candidates should include source evidence or state that evidence is missing.
