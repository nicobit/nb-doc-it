# License and Source Notes

This package is an original BMAD-close adapter for Microsoft Copilot Studio. It does not include the BMAD runtime or exact BMAD prompt files.

If your organization wants to import selected BMAD files into SharePoint knowledge, review the BMAD repository license and trademark notes first, then record imported files in `04-bmad-source-import` or your internal register.

Recommended naming for enterprise use:

- BRS to BMAD Handoff Analyst
- BMAD-Close Delivery Handoff Agent
- Delivery Handoff Orchestrator

Avoid implying that this adapter is the official BMAD product unless your organization has reviewed the naming and attribution implications.
