# Repo Target Structure

Copy the SharePoint handoff package into the repository like this:

```text
your-repo/
  docs/
    bmad-handoff/
      01-project-brief.md
      02-capability-map.md
      03-prd-draft.md
      04-epic-candidates.md
      05-story-candidates.md
      06-architecture-context-draft.md
      07-risks-assumptions-questions.md
      08-bmad-vscode-handoff-prompt.md
      09-handoff-readiness-check.md
```

BMAD should then validate/refine and decide where final PRD, architecture, and stories belong according to the installed BMAD project structure.
