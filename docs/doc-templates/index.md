# Templates

Use these templates when creating new documentation pages. Each template includes the required frontmatter metadata and a recommended content structure.

| Template | Use for |
|---|---|
| [Metadata](metadata-template.md) | Reference for all available frontmatter fields |
| [Feature](template-feature.md) | Documenting a product feature or module |
| [Runbook / SOP](template-runbook.md) | Operational runbooks and standard operating procedures |
| [ADR](template-adr.md) | Architecture Decision Records |
| [How-to](template-how-to.md) | Step-by-step how-to guides |
| [Service](template-service.md) | Service-level documentation in the module registry |
