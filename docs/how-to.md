## Contribution Rule

When adding a new how-to article:

1. Put it under the section where the work belongs (e.g. `operations/how-to/`, `engineering/how-to/`).
2. Use the [how-to template](doc-templates/template-how-to.md).
3. Add a link to this catalog under the matching heading.
4. Set `owner`, `status`, `next_review_date`, `classification`, and `criticality` in the frontmatter.
