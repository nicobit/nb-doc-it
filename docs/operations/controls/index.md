# Operational Controls

Use this area for operational checks and control activities that teams must perform consistently.

## Index

| Control | Frequency | Owner | Last performed |
|---|---|---|---|
| _Example: Release evidence check_ | Per release | _Release manager_ | _Date_ |

## Typical Controls

- Release evidence checks
- Access verification steps
- Change approval checkpoints
- Backup or restore verification
- Periodic operational reviews

When a control exists mainly for audit or compliance evidence, link to the relevant page in [Governance](../../governance/index.md).
